=> primeiros passos, Configuração 
git config --global user.name
git config --global user.email 

=> Inicialização do Repositorio
git init 

=> add arquivos e salvar 
git add . 
git commit -m "first commit"


=> add Repositorio remoto - Github

git remote add origin https://github.com/thiagodruvis/projeto_github.git


-> enviar 
git push -u origin main

=> seguir a triade 
git add . 
git commit -m "comandos"
git push
